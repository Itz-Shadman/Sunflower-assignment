# SkillSwap – A Local Skill Exchange Platform

## Project Overview
SkillSwap is an interactive platform that allows individuals to offer, learn, and trade skills within their local area. Users can browse skill listings, rate experiences, connect with providers, and book sessions. Whether it’s guitar lessons, language practice, coding help, or yoga training, SkillSwap makes local learning accessible and engaging.

## Key Features
- **User Authentication**: Email/password and Google login/signup.
- **Protected Routes**: Skill details and profile pages require authentication.
- **Skill Listings**: Dynamic display of skills from JSON data, including images, ratings, price, and description.
- **Book Sessions**: Users can book a skill session via a form with toast notifications.
- **Profile Management**: Users can update their name and photo.
- **Forgot Password**: Functional reset email system.
- **Responsive Design**: Fully responsive for mobile, tablet, and desktop.
- **Animations & Slider**: Hero slider with Swiper.js and smooth scroll animations with AOS.
- **SPA Behavior**: Single-page application with persistent Navbar and Footer.
